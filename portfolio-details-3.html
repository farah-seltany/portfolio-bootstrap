Portfolio administrable

Ce projet a été réalisé lors de ma dernière évaluation à la Wild Code School. C'est un portfolio qui a été réalisé en 24h.
Il est entièrement administrable depuis une page admin sécurisée par un formulaire d'authentification.
Il est possible depuis cette page de :

  • Modifier les données personnelles de l'utilisateur
  • Modifier les 3 qualités mises en avant
  • Ajouter, supprimer ou modifier les compétences
  • Ajouter, supprimer ou modifier les réalisations

Il est également possible de changer le mot de passe de connexion à la page admin.

TECHNOS : Angular 8, Node Js, Express Js, MySQL
